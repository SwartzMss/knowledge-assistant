# Knowledge Assistant

> **项目状态：** 本项目正在重构，接口、配置和部署方式可能随时调整，目前不建议用于生产环境。

Knowledge Assistant 是一个面向文档知识管理与问答的自托管应用。项目当前以最小可用基线为基础，逐步完成架构治理、测试补强和工程化改造。
